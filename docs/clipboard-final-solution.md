# 终端文本复制功能 - 最终方案

## 🎯 方案说明

为了避免 tmux copy-mode 导致的页面滚动和抖动问题，我们采用了**双模式**方案：

### 模式A：前端原生选择（推荐）

**特点**：
- ✅ **无滚动问题**
- ✅ **无抖动**
- ✅ 简单直观
- ✅ Mac 上支持 Cmd+C
- ❌ 无法在 vim/top 等 alternate screen 模式下使用

**操作步骤**：
1. **按住 Shift 键**
2. **鼠标拖选文本**（绕过 tmux mouse mode）
3. **按 Cmd+C** (Mac) 或 Ctrl+C (Windows/Linux)
4. ✅ 文本已复制到剪贴板

### 模式B：tmux copy-mode（高级用法）

**特点**：
- ✅ 支持在 vim/top 等 alternate screen 中选择
- ✅ 支持滚轮翻看历史
- ⚠️ 可能会有轻微滚动

**操作步骤**：
1. **手动进入 copy-mode**：按 `Ctrl+B` 然后按 `[`
2. **空格键开始选择**
3. **方向键或鼠标调整选择范围**
4. **按 Ctrl+C**：复制并清除选择（保持 copy-mode）
5. **按 ESC**：只清除选择，不退出
6. **按 q 或 Enter**：退出 copy-mode

---

## 🔧 配置变更

### 1. 鼠标拖选不再自动进入 copy-mode

**原因**：避免 tmux copy-mode 导致的页面滚动和抖动

**变更**：
- 之前：鼠标拖选 → 自动进入 copy-mode → 松开鼠标保持选择
- 现在：鼠标拖选 → **不进入 copy-mode**（需要手动进入）

### 2. 滚轮功能保留

**功能**：
- 命令行模式下：滚轮 → 进入 copy-mode 翻看历史
- vim/top 等应用中：滚轮 → 正常滚动应用内容

### 3. ESC 键行为优化

**变更**：
- 之前：ESC → 取消选择并退出 copy-mode
- 现在：ESC → **只清除选择，不退出 copy-mode**

## ⌨️ 快捷键总结

### tmux copy-mode 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B [` | 进入 copy-mode |
| `空格` | 开始选择 |
| `方向键` | 调整选择范围 |
| `Ctrl+C` | 复制 + 清除选择（保持 copy-mode） |
| `ESC` | 只清除选择（不退出） |
| `q` 或 `Enter` | 退出 copy-mode（不复制） |
| `滚轮` | 翻看历史（命令行模式） |

### 前端原生选择快捷键

| 快捷键 | 功能 |
|--------|------|
| `Shift + 鼠标拖选` | 选择文本 |
| `Cmd + C` (Mac) | 复制 |
| `Ctrl + C` (Win/Linux) | 复制 |

## 🎯 使用建议

### 场景1：普通命令行输出

**推荐**：使用 **前端原生选择**（Shift + 鼠标拖选）
- ✅ 最简单
- ✅ 无滚动问题
- ✅ Mac 上支持 Cmd+C

### 场景2：vim/top/less 等全屏应用

**推荐**：使用 **tmux copy-mode**
1. `Ctrl+B [`
2. 空格键开始选择
3. Ctrl+C 复制
4. q 退出

### 场景3：查看历史输出

**方法1**：滚轮翻看
- 在命令行模式下直接滚轮

**方法2**：tmux copy-mode
- `Ctrl+B [` 进入 copy-mode
- 方向键或 PageUp/PageDown 翻页
- q 退出

## ❓ 常见问题

### Q1: 为什么鼠标拖选不进入 copy-mode 了？

**A**: 为了避免 tmux copy-mode 导致的页面滚动和抖动问题。现在推荐使用 Shift+鼠标拖选的方式。

### Q2: 如何在 vim 中复制文本？

**A**: 两种方式：
1. 使用 vim 内置的复制命令（如 `yy` 复制整行）
2. 或者按 `Ctrl+B [` 进入 tmux copy-mode，选择后按 `Ctrl+C` 复制

### Q3: Mac 上 Ctrl 键在哪里？

**A**: Mac 键盘左下角的 `control` 键（不是 ⌘ 键）。

### Q4: 如果还是想用 tmux 鼠标选择怎么办？

**A**: 可以手动进入 copy-mode：
1. 按 `Ctrl+B [`
2. 用鼠标拖选
3. 按 `Ctrl+C` 复制
4. 按 `q` 退出
